# my-portfolio
A beginner portfolio website built with HTML and CSS showcasing my frontend skills.

## Technologies Used
- HTML5
- CSS3
- Flexbox
- CSS Grid
- Responsive Design
- CSS Variables
- CSS Transitions & Transform

## ✨ Features
- Fully responsive design — works on mobile and desktop
- Smooth navigation with fixed navbar
- About Me section
- Skills section
- Contact section

  ## 📂 Sections
- Home — Introduction and Hire Me button
- About — Background and description
- Skills — Technical skills
- Contact — Email and location
